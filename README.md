# Python 学习路线

从零开始系统学习 Python，覆盖核心语法到实际应用，记录完整的 Python 学习路线。

## 项目结构

```
py_project01/
├── 第一章/                  # 环境搭建与入门
├── 第二章/                  # 核心语法：数据、逻辑、容器、函数
├── 第三章/                  # 面向对象基础
├── 第四章/                  # 面向对象高级
├── 第五章/                  # 数据分析
├── 第六章/                  # 网络机器人
├── 第七章/                  # Web 应用
└── 第八章/                  # AI 应用
```

## 学习路线

### 第一阶段：核心语法

- **数据存储与运算** — 字面量、变量、常见数据类型、输入与输出、运算符
- **数据逻辑处理** — if 条件判断、match 模式匹配、while 循环、for 循环
- **数据存储容器** — list、str、tuple、set、dict 及其操作
- **函数** — 函数定义、参数、返回值、作用域、lambda 表达式
- **面向对象基础** — 类与对象、封装、继承、多态

### 第二阶段：应用实践

- **面向对象高级** — 魔术方法、描述符、元类、设计模式
- **数据分析** — NumPy、Pandas、Matplotlib、数据清洗与可视化
- **网络机器人** — HTTP 协议、requests、爬虫框架、反爬与自动化
- **Web 应用** — Flask / FastAPI、数据库、RESTful API、部署
- **AI 应用** — LLM API 调用、向量数据库、RAG、Agent 开发

## 环境

- Python 3.14
- 虚拟环境：`.venv/`

## 快速开始

```bash
git clone https://github.com/NagaSaki-SoyoLin/py_project01.git
cd py_project01
python -m venv .venv
# Windows
.venv\Scripts\activate
# macOS / Linux
source .venv/bin/activate
```
